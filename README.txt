Repo Git contenant l'exercice de Bataille Navale

Il est réalisé à l'aide de Django : le projet s'appelle bataille_navale, et l'app qui contient les codes s'appelle app_data.

Pour accéder au serveur, il faut éxécuter la commande :

python manage.py runserver

Il suffit alors de se rendre à l'adresse : 127.0.0.1:8000/app_data

On peut alors sélectionner la taille de la grille souhaitée et générer un placement de bateaux selon les contraintes de l'énoncé.
